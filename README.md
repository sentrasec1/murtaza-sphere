An educational mobile application built with Kotlin for Android devices.

## Overview

Murtaza Sphere is a mobile learning platform designed to provide an interactive and engaging educational experience. The application leverages Android technologies to deliver content-rich learning modules and interactive features for users of all ages.

## Features

- **Interactive Learning Modules** - Engaging educational content presented in an interactive format
- **Mobile-First Design** - Optimized for seamless mobile device experience
- **User-Friendly Interface** - Intuitive navigation and clear information architecture
- **Responsive Layout** - Adapts to various screen sizes and orientations

## Technology Stack

- **Language**: Kotlin
- **Platform**: Android
- **Target SDK**: Android (latest stable)

## Getting Started

### Prerequisites

- Android Studio (latest version recommended)
- Kotlin 1.x or higher
- Gradle build system
- Android SDK (API level 21 or higher)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/sentrasec1/murtaza-sphere.git
```

2. Open the project in Android Studio:
   - Select "Open an Existing Project"
   - Navigate to the cloned repository directory
   - Click "Open"

3. Build the project:
   - Go to Build → Make Project
   - Wait for Gradle to build the project

4. Run the app:
   - Connect an Android device or start an emulator
   - Click Run → Run 'app'
   - Select your device/emulator and click OK
